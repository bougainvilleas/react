# react study
##前后端分离项目
- 前端react
- 后台springboot1.5.x

##下载一些依赖
- PropType
```shell script
npm install --save prop-types
```
- axios
```shell script
npm install --save axios
```
- markdown
```shell script
npm install --save showdown
```
- bootstrap
```shell script
npm install --save bootstrap
```
- 发布订阅PubSubJS库，组件间通信
```shell script
npm install pubsub-js --save
```
